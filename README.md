# Proiect-RC2
Proiect realizat in Cisco pentru disciplina RC22 - Retele de Calculatoare

*--- Descriere ---*

Un proiect realizat in Cisco pentru disciplina RC2 - Retele de Calculatoare, unde am realizat o simulare a unei retele. Reteaua este structurata pe mai multe segmente si include harta logica, harta fizica si harta a serviciilor, documentate in Visio. 

Proiectul acopera definirea completa a adresarii IP (retele, masti, gateway-uri, nr. de hosturi), proiectarea topologiei fizice cu evidentierea porturilor pentru fiecare legatura si configurarea echipamentele de retea (rutere si switch-uri).

De asemenea, sunt implementate si configurate servicii esentiale de retea (DHCP, DNS, HTTP, FTP, MAIL), gazduite intr-un N.O.C; avem incluse si servicii de partajare a resurselor (Data Base servers, imprimante, cu restrictii de access pe retea).

Proiectul include documentatie tehnica completa (configuratii echipamente si servere, capturi de ecran, precum si un tabel de estimare a costurilor prin cablare, comutare, rutare si servicii).

*--- Scopul proiectului ---*

Scopul principal este simularea unei infrastructuri reale de retea si intelegerea modului de functionare si interconectare a componentelor acesteia.

*--- Informatii utile ---*
Proiectul poate fi verificat direct in Cisco Packet Tracer, iar documentatia completa (partea scrisa) este disponibila pentru consultare.
